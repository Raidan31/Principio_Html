Atributos Novos no HTML 5

autofocus Ao abrir o site o usuário será direcionado para o elemento que tiver este atributo

placeholder Cria aqueles exemplos no elemento que tiver o atributo

required Torna os campos obrigatórios

disabled Por meio do fieldset permite desabilitar todos os campos

maxlength Define o numero maximo de caracteres a serem escritos na textarea, exclusivo do textarea

minlength Define o numero minimo de caracteres a serem escritos na textarea, presente em todos os inputs

form Permite utilizar um input fora do campo form. ex:

contenteditable Permite tornar o elemento selecionado em um campo editavel
